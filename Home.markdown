# BMS Library

## Information

1. This website distributes BMS song and sabun packages.
2. Insane packages are intended for experienced BMS players.
3. Ordinary level-scale packages are not planned.
4. Some songs or sabuns may be missing because of hash mismatches or unavailable source data.
5. Packages are usually renewed once a year, with patches released every few months.

Latest fix: 2025-12-24

## Download Notes

- Please use the torrent files to download large packages.
- If seeding or peer discovery does not work in your torrent client, try another torrent client before reporting the problem.
- Download speed and availability may vary depending on the current seed status.
- Some packages now include Solar / Supernova content, so please make sure you have enough free space before downloading.
